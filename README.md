# Prova de Álgebra Linear Computacional - UFRRJ

Repositório com implementações práticas de Álgebra Linear Computacional, incluindo classificação KNN. Desenvolvido a partir da resolução de uma prova prática da disciplina na UFRRJ. Em cada questão existe uma documentação própria contendo introdução, decisões arquiteturais de software, uso de IA e testes realizados.

**Nota**: Para rodar o programa, certifique-se de carregar os arquivos (.xlsx) no ambiente de sessão do Colab antes de executar as células de leitura de dados.

## Tecnologias Utilizadas
* **Linguagem:** Python
* **Ambiente:** Google Colab
* **Bibliotecas Principais:** NumPy, Pandas, Scikit-Learn (para o KNN)
